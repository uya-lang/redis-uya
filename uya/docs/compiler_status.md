## 语言特性支持

### ✅ 核心特性（已实现）

- [x] **基本类型**: i8, i16, i32, i64, u8, u16, u32, u64, f32, f64, bool, byte, void
- [x] **指针类型**: `&T` (普通指针), `*T` (FFI 指针)
- [x] **数组类型**: `[T: N]` (固定大小数组)
- [x] **切片类型**: `&[T]`, `&[T: N]`
- [x] **结构体**: 普通结构体、extern struct
- [x] **联合体 (union)**: 编译期标签跟踪
- [x] **枚举 (enum)**: C 风格枚举
- [x] **元组**: `(T1, T2, ...)`
- [x] **错误联合类型**: `!T`
- [x] **泛型**: `<T>`, `<T: Constraint>`
- [x] **接口 (interface)**: 鸭子类型
- [x] **方法块 / 方法调用**: 结构体与联合体方法、`Type.method(...)`、`obj.method(...)` 语法糖、泛型方法与无限制链式后缀调用
- [x] **函数**: 普通函数、泛型函数、可变参数函数
- [x] **控制流**: if/else, while, for, break, continue, return
- [x] **defer/errdefer**: 作用域结束执行
- [x] **match 表达式**: 模式匹配
- [x] **try/catch**: 错误处理
- [x] **字符串插值**: `"text${expr}"`
- [x] **切片语法**: `&arr[start:len]`
- [x] **指针算术**: `ptr +/- offset`

### ✅ 内存安全特性（已实现）

- [x] 数组边界检查
- [x] 空指针检测
- [x] 未初始化变量检测
- [x] 整数溢出检测
- [x] 除零检测
- [x] 移动语义检测
- [x] 内存安全证明系统

### ✅ 内置函数（已实现）

- [x] `@size_of(T)` - 类型大小
- [x] `@align_of(T)` - 类型对齐
- [x] `@len(arr)` - 数组长度
- [x] `@max(T)` / `@min(T)` - 类型极值
- [x] `@embed("path")` - 编译期嵌入单文件
- [x] `@embed_dir("path")` - 编译期递归嵌入目录
- [x] `@syscall(nr, ...)` - 系统调用
- [x] `@va_start` / `@va_arg` / `@va_end` / `@va_copy` - 可变参数（va_list 为内置类型）
- [x] `va_list` 内置类型
- [x] `@src_name` / `@src_path` / `@src_line` - 源码位置
- [x] `@src_col` / `@func_name` - 源码位置与函数名
- [x] `@ptr_from_usize` / `@usize_from_ptr` - 指针转换

### ✅ 宏系统（已实现）
